# 仕様変更 ステップ1

- 空き地を新たに購入したので（parking_space）を追加したい。新たな空き地は5区画あります
- 支払いの履歴を残し、年払いに半年払いなどにも対応したい
- 3ヶ月連続で未払の駐車場の契約者を抽出したい

# 仕様変更 ステップ2
1. 時間貸しの導入
1. 自動車以外への貸出


## 時間貸しの導入
parking_spaceは月極だけだったが、時間貸しも行う。
月極契約ないparking_spaceを時間貸しに利用することで空きスペースの最適化を行う。

1. 10分間100円とし、8:00-19:59の間の最大連続利用料は5000円とする
1. 20:00 - 7:59 の間の最大利用料は2000円とする
1. 8:00または20:00をまたいだ場合は最大連続時間はリセットされ、切り替え前の支払額と切り替え後のそれぞれの支払額の合算を請求する
1. 契約終了した空きparking_spaceは即日時間貸しとして利用できる

## 自動車以外への貸出
暗黙的に4輪自動車を対象にしていたが、自動2輪への貸出も追加する。
この場合、1つのparking_spaceに対して車は1台だが自動2輪は3台駐車することができる

自動2輪は月極、時間貸し共に対象である。
